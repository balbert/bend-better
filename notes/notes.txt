    pip install pylint autopep8
        or

Configure VS Code Settings
--------------------------

●   Open settings (File > Preferences > Settings, or Ctrl+,).

●   Search for "Python Linting" and "Python Formatting" settings.

●   Enable Linting:

    Ensure python.linting.enabled is set to true.

●   Select Linter:

    Choose your preferred linter (e.g., Pylint, Flake8, Ruff) by setting
    python.linting.pylintEnabled, python.linting.flake8Enabled,
    or configuring the Ruff extension.

●   Select Formatter:

    Set python.formatting.provider to your chosen formatter (e.g., autopep8, black, ruff).

●   Format on Save:

    To automatically format your code when saving, add the following to your settings.json:

    "[python]": {
        "editor.defaultFormatter": "ms-python.autopep8", // or "ms-python.black", "charliermarsh.ruff"
        "editor.formatOnSave": true
    }

    Adjust editor.defaultFormatter to match your chosen formatter.
